🧠 Reflection & Learning Journey

---

### What I Learned
DOM Manipulation: I practiced creating elements dynamically and updating the UI based on user actions.

Data Persistence: Working with JSON.parse and JSON.stringify taught me how to handle data in the browser.

Responsive Design: I learned how to use breakpoints (600px) to ensure my code looks great on mobile devices.

Grid vs. Flexbox: I now understand that Flexbox is for alignment in a row, while Grid is for structural layouts.

---

### Challenges
State Management: Keeping the "Completed" status synced with LocalStorage was a challenge. I had to move from storing simple strings to storing objects.

Specificity: Debugging CSS when media queries weren't firing taught me a lot about selector priority.
